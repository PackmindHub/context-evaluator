# Show Remediation Strategy in Action Summary

## Context

The suggestion enrich prompt already asks the AI to include an `outputType` field (`"standard"`, `"skill"`, or `"generic"`) in its JSON response for each addressed suggestion. However, the summary parser drops this field, the types don't include it, and the UI doesn't display it. The goal is to flow `outputType` through the full stack so it appears as a badge in the Action Summary section of the Remediation History card.

Only **suggestions** get a strategy badge — error fixes edit existing content and don't involve a strategy decision.
